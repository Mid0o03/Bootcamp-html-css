# 01 - Profile Card

Un composant de carte de profil responsive et élégant.

## Aperçu
![Aperçu du projet](preview.png)

## Description
Ce projet est un défi d'intégration HTML/CSS visant à créer une carte de profil utilisateur moderne. Il met en pratique :
- La structuration sémantique HTML5 (`article`, `header`, `section`)
- Le positionnement avec Flexbox
- Le stylage des images (object-fit, border-radius)
- L'utilisation de pseudo-éléments et d'ombres portées

## Technologies
- HTML5
- CSS3

## Installation
Ouvrez simplement le fichier `index.html` dans votre navigateur.
